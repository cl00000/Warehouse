使用 SSH 替代 HTTPS
    如果 HTTPS 连接有问题，可以改用 SSH 协议：
    确保你已添加 SSH 密钥到 GitHub（参考 GitHub SSH 文档）。
    将远程仓库 URL 改为 SSH 格式：
    git remote set-url origin git@github.com:cl00000/HKR.git
然后再次尝试推送：
    git push


生成密钥对
    ssh-keygen -t ed25519 -C "787990099@QQ.com"
密钥对保存路径
    C:\Users\78799/.ssh/id_ed25519
        私钥：id_ed25519
        公钥：id_ed25519.pub
将公钥内容输出到屏幕
    cat ~/.ssh/id_ed25519.pub
